Example app

To kick of the process:

curl -X GET -H "Content-Type: application/json" -H "Cache-Control: no-cache" 'http://localhost:9999/example/css/reset'

Following requests need to satisfy the requirements returned from the previous request, like so:

curl -X POST -H "Content-Type: application/json" -H "Cache-Control: no-cache" -d '{ "input": { "username": "andy123" } }' 'http://localhost:9999/example/css/reset?_action=submitRequirements'

When a token is returned in the response along with requirements, it must be echoed back like so:

curl -X POST -H "Content-Type: application/json" -H "Cache-Control: no-cache" -d '{ "token": "eyAidHlwIjogIkpXVCIsICJhbGciOiAiSFMyNTYiIH0.eyAic3RhZ2VUYWciOiAidmFsaWRhdGVDb2RlVGFnIiwgInN0YWdlSW5kZXgiOiAiMCIsICJ1c2VySWQiOiAiYW5keTEyMyIsICJjb2RlIjogImRhNzRmNDVjLTYzNjgtNDE0NC04NDU2LTBkOGZlODFjMjE1NSIgfQ.0jIOiB6QXbrAKpPbDNrvXp72WrOozOiHQfQwWf60USk", "input": { "code": "da74f45c-6368-4144-8456-0d8fe81c2155" } }' 'http://localhost:9999/example/css/reset?_action=submitRequirements'
